OPS-L Standards

A guide to animal production data specifications

Overview

The specification provides a framework for modeling and sharing data related to animal production. It includes definitions for common entities, events, and data-sharing mechanisms to ensure interoperability and consistency across systems.

Contents

  • Github Work Site - Provides an overview of the standard and its purpose, how to contribute.
  • Logical Animal Information Model (LAIM) - Defines the core entities and relationships in the model.
  • Data Sharing - Describes the mechanisms for sharing data using APIs and RDF representations.
